Programación Lineal - Ejemplo Resuelto

✅ Sistema de Restricciones (S.A.)

Maximizar: Z = 60x + 80y
Sujeto a:
4x + 2y ≤ 100
2x + 3y ≤ 90
x ≥ 0, y ≥ 0

✅ Despejes paso a paso (en función de y)

1) 4x + 2y ≤ 100
→ 2y ≤ 100 - 4x
→ y ≤ (100 - 4x)/2
→ y ≤ 50 - 2x

2) 2x + 3y ≤ 90
→ 3y ≤ 90 - 2x
→ y ≤ (90 - 2x)/3
→ y ≤ 30 - (2/3)x

✅ Intersección desarrollada (igualando las y)

Igualamos:
50 - 2x = 30 - (2/3)x
Multiplicamos todo por 3:
150 - 6x = 90 - 2x
→ 60 = 4x → x = 15
Sustituimos en y:
y = 50 - 2(15) = 20
Intersección: (15, 20)

✅ Conclusión

El valor máximo de Z = 60x + 80y se alcanza en el punto (15, 20), y es Z = 3000. ✅
Por lo tanto, la solución óptima es producir 15 unidades de x y 20 unidades de y.
